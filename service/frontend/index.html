ATRIUM Page Classification

Historical document classification service using deep learning.

This service provides a FastAPI interface for the Atrium Page Classification models. It allows users to upload document images or PDFs and receive structural classification predictions (e.g., Text, Drawing, Table) using various fine-tuned deep learning models.

Supported Models 🧠

Version Base Architecture Best Accuracy Notes
v4.3 RegNetY 160 99.17% Balanced option. Best performing "Small" model.
v5.3 ViT-Large (384) 99.12% Largest model, highest accuracy, slowest inference.
v2.3 / v3.3 ViT-Base (224/384) ~98.8% Standard Transformer baselines.
v1.3 EfficientNetV2-M 98.83% CNN-based, faster inference.

Categories 🪧

TEXT_P 📄 Printed: Published paragraphs.
TEXT_HW ✏️📄 Handwritten: Handwritten paragraphs.
LINE_T 📏 Typed Table: Machine-typed tabular structure.
LINE_P 📏 Printed Table: Printed tabular structure.
LINE_HW ✏️📏 Handwritten Table: Handwritten tabular structure.
DRAW 📈 Drawing: Maps, paintings, schematics.
DRAW_L 📈📏 Structured Drawing: Drawings in a layout/legend.
PHOTO 🌄 Photo: Photographs.

API Endpoints 🔗

Method Path Description
GET / Serves this HTML interface.
GET /info Returns metadata about available models and device.
POST /predict_image Inference for single image (JPG/PNG).
POST /predict_document Inference for PDF documents (multipage).

Request Example (Image) 💻

POST /predict_image

Form Data:

  • file: Binary image (JPG/PNG).
  • version: e.g., v5.3 or all.
  • topn: Integer (e.g., 3).
