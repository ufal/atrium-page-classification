Category Descriptions

The model classifies pages into 11 distinct categories based on content (Text, Photo, Drawing) and layout (Paragraph vs. Table/Linear). This sorting allows for content-specific downstream processing (e.g., OCR vs. Image Extraction).

Label Description
TEXT 📰 Mixed Text: Mixtures of printed, handwritten, or typed text, potentially with minor graphical elements.
TEXT_T 📄 Typed: Only machine-typed text in paragraph or block form.
TEXT_P 📄 Printed: Only printed text in paragraph or block form.
TEXT_HW ✏️📄 Handwritten: Only handwritten text in paragraph or block form.
LINE_T 📏 Typed Table: Machine-typed text organized in a tabular or form-like structure.
LINE_P 📏 Printed Table: Printed text organized in a tabular or form-like structure.
LINE_HW ✏️📏 Handwritten Table: Handwritten text organized in a tabular or form-like structure.
DRAW 📈 Drawing: Maps, paintings, schematics, or graphics (potentially with captions).
DRAW_L 📈📏 Structured Drawing: Drawings presented within a table-like layout or legend.
PHOTO 🌄 Photo: Photographs or photographic cutouts.
PHOTO_L 🌄📏 Structured Photo: Photos presented within a table-like layout.

Model Versions

Models are fine-tuned on a dataset of ~48,000 archival pages. Choose a model based on your speed vs. accuracy requirements.

Version Base Architecture Best Accuracy Notes
v5.3 ViT-Large (Patch16-384) 99.12% Largest model, highest accuracy, slowest inference.
v4.3 RegNetY 160 99.17% Balanced option. Best performing "Small" model.
v1.3 EfficientNetV2-M 98.83% CNN-based, faster inference, good for smaller hardware.
v2.3 / v3.3 ViT-Base (224/384) ~98.8% Standard Transformer baselines.
